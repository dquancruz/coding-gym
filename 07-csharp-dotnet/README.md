# 09 — C# / .NET

**Rol en el plan:** Mantenimiento (1 ejercicio/semana rotando) desde el día uno.
Sube a secundario en Fase 2 si tu objetivo es enterprise Microsoft-stack.

## Checklist Junior → Mid

- [ ] LINQ (consultas sobre colecciones, estilo declarativo)
- [ ] async/await correctamente (evitar deadlocks comunes)
- [ ] Genéricos e interfaces, programación contra abstracciones
- [ ] ASP.NET Core: minimal APIs o controladores MVC
- [ ] Entity Framework Core: migraciones, queries, evitar N+1
- [ ] Inyección de dependencias (built-in de .NET)
- [ ] xUnit para tests, mocks con Moq/NSubstitute
- [ ] Records y pattern matching (C# moderno)
- [ ] Manejo de configuración por ambiente (`appsettings.json`)

## Señal de que ya eres mid en este track
Tu código C# usa LINQ y async de forma idiomática, no traducido de
imperativo, y tu API maneja errores con `ProblemDetails` consistente en vez
de excepciones sin capturar.
