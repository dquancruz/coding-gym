# 09 — C# / .NET

**Role in the plan:** Maintenance (1 exercise/week rotating) from day one.
Bumps to secondary in Phase 2 if your goal is Microsoft-stack enterprise.

## Junior → Mid Checklist

- [ ] LINQ (queries over collections, declarative style)
- [ ] Correct async/await (avoiding common deadlocks)
- [ ] Generics and interfaces, programming against abstractions
- [ ] ASP.NET Core: minimal APIs or MVC controllers
- [ ] Entity Framework Core: migrations, queries, avoiding N+1
- [ ] Dependency injection (built into .NET)
- [ ] xUnit for tests, mocks with Moq/NSubstitute
- [ ] Records and pattern matching (modern C#)
- [ ] Per-environment configuration handling (`appsettings.json`)

## Sign that you're already mid on this track
Your C# code uses LINQ and async idiomatically, not translated from
imperative style, and your API handles errors with consistent
`ProblemDetails` instead of uncaught exceptions.
