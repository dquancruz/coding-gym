# 04 — Python

**Rol en el plan:** Primario en Fase 2. Puente directo hacia IA/ML.

## Checklist Junior → Mid

- [ ] Type hints en funciones y variables (`mypy` sin errores)
- [ ] `dataclasses` / Pydantic models en vez de diccionarios sueltos
- [ ] Comprehensions idiomáticas (lista/dict/set) sin abusar de ellas
- [ ] Generadores e iteradores para datos grandes
- [ ] Context managers (`with`) propios, no solo usarlos
- [ ] Manejo de excepciones específico (no `except Exception` genérico)
- [ ] Módulos y paquetes, imports relativos vs absolutos
- [ ] Entornos virtuales y gestión de dependencias (`venv`/`poetry`)
- [ ] `pytest`: fixtures, parametrize, mocks
- [ ] FastAPI: path/query params, Pydantic, dependency injection
- [ ] Decoradores propios (nivel mid-avanzado)
- [ ] Async en Python (`asyncio`) cuando aplica

## Señal de que ya eres mid en este track
Tu código Python se lee como Python (idiomático), no como Java o JS traducido
línea a línea. Usas type hints como documentación viva, no como decoración.
