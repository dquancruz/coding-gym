# 🏋️ Coding Gym

Repositorio de práctica activa para ir de **junior a mid** en un stack
full-stack + IA, usando Claude Code como entrenador.

## Cómo funciona
1. `CLAUDE.md` define las reglas del juego (lee esto primero si eres Claude Code).
2. `ROADMAP.md` define el stack completo y la fase activa del plan.
3. Cada carpeta `NN-track/` tiene su propio checklist junior→mid en su `README.md`.
4. `progress/tracker.md` y `progress/log.md` llevan el registro de avance.

## Uso diario
Abre Claude Code en esta carpeta y escribe:

```
ejercicio
```

Claude elegirá el track y skill según la fase activa y tu tracker, y generará
un ejercicio en `NN-track/exercises/`. Resuélvelo, luego:

```
revisar
```

Otros comandos: `progreso`, `subir nivel [skill]`, `siguiente fase`.
